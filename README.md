# Flask portfolio website
## Description
This is a portfolio website I made with Flask for the assignment on Day 83 of the 100 Days of Code course. 


## Installation
```bash
git clone https://github.com/wnsnk/flask-portfolio-website
cd flask-portfolio-website
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
