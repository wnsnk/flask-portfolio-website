# Flask portfolio website
## Description
This is a portfolio website I made with Flask for the assignment on Day 83 of the 100 Days of Code course. 


## Installation
- Clone repository, create virtual environment and install dependencies:
```bash
git clone https://github.com/wnsnk/flask-portfolio-website.git
cd flask-portfolio-website
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

- create a .env file and add environment variables (see .env.example)

- run the app
```bash
python main.py
```

flask will start a local server where the app is hosted.

- in your webbrowser go to: http://127.0.0.1:5000
